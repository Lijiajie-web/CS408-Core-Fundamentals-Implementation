## 📌 About This Repository
Welcome to my academic code repository. This repository contains my independent implementations of core computer science fundamentals. 

These codes were deeply optimized and practiced during my rigorous preparation for the **China CS408 Graduate Entrance Examination** (the most challenging CS exam in China, covering Data Structures, Computer Organization, Operating Systems, and Computer Networks). 

As a student deeply passionate about **Discrete Mathematics** and **Number Theory**, I believe that writing foundational code from scratch—rather than relying on high-level APIs—is the best way to understand the mathematical soul of Computer Science.
